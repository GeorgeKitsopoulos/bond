# Public Repository Candidate

This directory is a clean public candidate export of Bond.

It was created from the private/local development repository without preserving private git history.

The public repository must not contain:

- local runtime data
- private assistant memory
- private configuration
- secrets or credentials
- Gitea runtime data
- virtual environments
- caches
- local machine paths
- assistant/operator prompt history

The source private repository remains the historical development workspace. This candidate is intended to become the first public GitHub history if final safety checks pass.
